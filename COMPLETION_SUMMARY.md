### Option 1: Railway (Recommended)

1. **Create Railway Project**
   ```bash
   # Railway will auto-detect the GitHub repository
   # Connect to dioara/explorecotswolds
   ```

2. **Add MySQL Database**
   - Add MySQL plugin in Railway
   - Database URL will be auto-configured

3. **Configure Environment**
   ```bash
   # Railway will use the DATABASE_URL from MySQL plugin
   # No additional configuration needed
   ```

4. **Deploy**
   ```bash
   # Railway will automatically:
   # - Run pnpm install
   # - Run pnpm build
   # - Start the server
   ```

5. **Seed Database**
   ```bash
   # After first deployment, run in Railway console:
   pnpm db:push
   pnpm tsx server/cotswolds-seed.ts
   pnpm tsx server/seed-blog.ts
   ```

6. **Configure Domain**
   - Add custom domain: explorecotswolds.com
   - Update DNS records as instructed by Railway

### Option 2: Vercel + PlanetScale

1. **Create Vercel Project**
   - Import from GitHub: dioara/explorecotswolds
   - Framework: Vite
   - Build command: `pnpm build`
   - Output directory: `dist/public`

2. **Create PlanetScale Database**
   ```bash
   pscale database create explorecotswolds
   pscale connect explorecotswolds main
   ```

3. **Configure Environment Variables**
   ```
   DATABASE_URL=mysql://[connection-string]
   ```

4. **Deploy and Seed**
   ```bash
   pnpm db:push
   pnpm tsx server/cotswolds-seed.ts
   pnpm tsx server/seed-blog.ts
   ```

5. **Configure Domain**
   - Add explorecotswolds.com in Vercel dashboard

```bash

# Configure database
```
